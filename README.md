### Prerequisites
- Node.js 18+

### Installation

```bash
# Clone repository
git clone https://github.com/amansingh4517/aegis-file-verifier.git
cd aegis-file-verifier

# Backend setup
cd backend
npm install
cp .env.example .env
# Add your GEMINI_API_KEY, MONGODB_URI, JWT_SECRET to .env

# Frontend setup
cd ../frontend
pnpm install
cp .env.example .env.local
# Add NEXT_PUBLIC_API_URL=http://localhost:5000

# Run application
# Terminal 1 - Backend
cd backend && npm start

# Terminal 2 - Frontend  
cd frontend && pnpm dev
```
